Provenance for The Communications Protocols Controlled Vocabulary was created for NSF's EarthCube Program's Resource Repository. At this point it merely lists the protocols used by these resources, with the eventual goal of enabling the more complete descriptions that would be required to improve interoperability between resources such as data sets and software tools.
